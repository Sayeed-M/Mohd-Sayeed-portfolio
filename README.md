## 🚀 Overview

Welcome to the source code for my interactive portfolio. This platform was built to serve as a **dynamic telemetry terminal**, showcasing my work in building systems that process the world in real-time—from deep embedded flight controllers to production-grade mobile applications.

It features a custom **dark/light mode glassmorphic UI**, interactive WebGL simulations (like the VTOL Drone Hover simulation), an integrated AI Core chat module, and a full dashboard administration system built entirely from scratch.

---

## 🛠️ Tech Stack Architecture

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS & Vanilla CSS Variables
- **Animations**: Framer Motion (Scroll parallax, springs, dynamic viewport entries)
- **Simulations**: HTML5 Canvas & WebGL Frame Players
- **Database**: Custom JSON routing mapping system (no external backend required)
- **Icons**: Lucide React & Native SVGs

---

## 🧠 Core Features

* **Cinematic Hero**: Scroll-driven frame sequences rendering 3D aerospace assets directly in the browser runtime.
* **Integrated AI Terminal**: A mock neural network interface mapped over my resume logic.
* **Interactive Sandboxes**: Live feedback pipelines of quadplane transitional hovering states, displaying target lock-on vectors.
* **Global Edit Architecture**: A hidden admin pane allowing optimistic UI updates to the static JSON cluster on the fly. 
* **Dynamic Dark Mode**: Absolute state toggle forcing HEX `#0a0a0a` over UI clusters for immediate theme inversions.
